    About the Router Simulator:
    This simulator represents routers as nodes in a network. It's designed to help visualize and understand how routers connect and communicate in a network topology.

    How it works:

    • Click on "Add Router" button to add a new router.
    • Click on a Router to make connections.
    • Input the Router name and weight .
    • Use the "Find Path" button to find the shortest path between source and destination provided as input by you.
    • Click the log icon to toggle connection and path history.
    • Use "Reset" button to reset the network simulation.
